# Mobile Sales Dashboard – Power BI

## Project Overview

This project is an interactive Mobile Sales Dashboard developed using Microsoft Power BI. The dashboard provides insights into mobile phone sales performance by analyzing sales, quantity sold, transactions, customer ratings, payment methods, and regional performance.

The dashboard helps users visualize business data and make data-driven decisions through interactive charts, maps, and filters.

---

## Objectives

* Analyze mobile sales performance
* Track total sales, quantity, and transactions
* Compare sales across different cities and brands
* Monitor customer ratings
* Analyze payment method usage
* Visualize monthly sales trends

---

## Tools Used

* Microsoft Power BI
* Microsoft Excel

---

## Dataset Features

The dataset includes:

* Mobile Model
* Brand
* City
* Customer Name
* Customer Age
* Payment Method
* Quantity Sold
* Total Sales
* Transactions
* Customer Ratings
